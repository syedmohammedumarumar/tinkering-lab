# ✅ **EXPERIMENT 1 – FINAL EXAM WRITE-UP**

---

## 🎯 **Aim**

To demonstrate **series and parallel circuits** on a breadboard using LEDs.

---

## 🔧 **Components Required**

* Breadboard
* 9V Battery with clip
* LEDs (Red, Yellow, Green)
* Resistors (470Ω)
* Jumper wires

---

## 📘 **Theory (VERY SHORT – WRITE THIS ONLY)**

* In a **series circuit**, current is same through all components and total resistance:
  **R = R1 + R2 + R3**
* In a **parallel circuit**, voltage is same across all components and:
  **1/R = 1/R1 + 1/R2 + 1/R3**
* LEDs glow to indicate current flow. Brightness depends on voltage.

👉 (Optional line if you want extra marks)
Different LEDs can indicate battery levels (Green–High, Yellow–Medium, Red–Low).

---

## ⚙️ **Procedure (SHORT & CLEAN)**

### 🔹 Series Circuit:

1. Connect battery + to breadboard positive rail
2. Connect LED1 anode to + rail
3. Connect LED1 cathode → resistor
4. Connect resistor → next LED
5. Repeat for all LEDs
6. Last LED → negative rail
7. Check LEDs glow

---

### 🔹 Parallel Circuit:

1. Connect all LED anodes to + rail
2. Connect each LED cathode → separate resistor
3. Connect resistors to – rail
4. Check LEDs glow

---

## 💻 **Code**

👉 **No code required** (Important – write this if needed)

---

## 📊 **Output**

* In **series**: LEDs glow but brightness is less
* In **parallel**: LEDs glow brighter

---

## ✅ **Result**

Thus, **series and parallel circuits were successfully implemented on a breadboard using LEDs**.
